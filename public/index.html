Error Relativo y Error Absoluto

En la medición y el análisis de datos, es común encontrar dos conceptos clave: el error absoluto y el error relativo. Estos conceptos nos ayudan a entender cuán precisas son nuestras mediciones o cálculos.

Error Absoluto

El error absoluto se define como la diferencia entre el valor medido o calculado y el valor verdadero o aceptado.

Error Absoluto = | Valor medido - Valor verdadero |

Error Relativo

El error relativo compara el error absoluto con el valor verdadero o aceptado. Se expresa generalmente como un porcentaje.

Error Relativo = (Error Absoluto / Valor verdadero) × 100%

Ejemplo:

Supongamos que estamos midiendo la longitud de un objeto. El valor verdadero es 10 cm, pero nuestra medición nos da 10.2 cm.

Error Absoluto = |10.2 - 10| = 0.2 cm

Error Relativo = (0.2 / 10) × 100% = 2%

Por lo tanto, nuestra medición tiene un error relativo del 2%.
